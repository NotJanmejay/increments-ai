1. Navigate to client `cd client`
2. Run `npm install` in client directory
3. Run `npm run dev` to start local server

4. From a different terminal navigate to server directory using `cd server`
5. Create a virtual environment using `python -m venv .venv` and activate using `.venv\Scripts\activate` (in windows)
6. Run `pip install -r requirements` to install python directory
7. In .env file fill the appropriate keys
8. Run `python manage.py runserver` to start backend server

9. Visit localhost:5173 for development website
10. For Teacher route visit -> /teacher
11. For Student Chatbot -> /student
